# ⚡ Gamified Task Manager

A simple and interactive productivity web application built using React.
This project helps users manage daily tasks while staying motivated through a gamification system like XP, levels, and streaks.

## 📌 Problem Statement

Students often struggle to stay consistent with their tasks and productivity.
Traditional to-do apps lack motivation and engagement.

This project solves this problem by combining task management with gamification elements like XP, levels, and streaks to keep users motivated.

---

## 🎯 Objectives

- To build a simple task management system
- To implement gamification for motivation
- To use React concepts like hooks and context
- To integrate API for real-time learning tips
- To create a clean and user-friendly UI

---

## 🚀 Features

### 📝 Task Management

- Add new tasks
- Delete tasks
- Mark tasks as completed

### 🎮 Gamification System

- XP (points) system
- Level progression
- Daily streak tracking

### 💡 Productivity Tips

- Fetch tips from Wikipedia API
- Random tip generation
- Copy tip feature
- Save tip to localStorage

### ⭐ Saved Tips

- View saved tips
- Delete saved tips

### 🎨 UI Features

- Clean responsive design
- Dark mode toggle
- Navbar with XP display

### ⚡ Performance

- Lazy loading
- Debounce optimization

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **State Management:** Context API
- **API:** Wikipedia API
- **Storage:** LocalStorage

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/arpitpaatni2007-jpg/gamified-task-manager

2. Navigate to project folder:
   cd gamified-task-manager

3. Install dependencies:
   npm install

4. Run project:
   npm run dev
